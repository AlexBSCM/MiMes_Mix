# 🚀 MiMes Mix

Современный мессенджер с поддержкой чатов, звонков и управления профилем.

[![Android CI](https://github.com/AlexBSCM/MiMes_Mix/actions/workflows/android-ci.yml/badge.svg)](https://github.com/AlexBSCM/MiMes_Mix/actions/workflows/android-ci.yml)

## 🛠️ Стек технологий
- **Android:** Kotlin, Jetpack Compose, Hilt, MVVM
- **Backend:** Node.js, TypeScript, Prisma
- **Frontend:** Expo / React Native

## ⚙️ Установка и запуск
1. Установите Android Studio, Node.js и настройте Firebase.
2. Положите файл google-services.json в папку pp/.
3. Backend: cd backend -> 
pm install -> 
pm run dev
4. Frontend: cd frontend -> 
pm install -> 
px expo start

## 🤝 CI/CD
Проект использует GitHub Actions для автоматической сборки APK при каждом push в ветку master. Готовые артефакты доступны во вкладке [Actions](https://github.com/AlexBSCM/MiMes_Mix/actions).
